ApiErrorCode

enum class ApiErrorCode

Enum Values

INVALID_CONFIG

Invalid configuration parameters were passed.

NOT_AUTHORIZED

User is not authorized to perform the operation requested.

SUDO_NOT_FOUND

Sudo being updated or deleted is not found.

VERSION_MISMATCH

The version of the resource that's being updated or deleted does not match the version stored in the backend.

SERVER_ERROR

An internal server error cause the API call to fail. The error is possibly transient and retrying at a later time may cause the call to complete successfully.

BAD_DATA

Indicates the bad data was found in cache or in backend response.

GRAPHQL_ERROR

GraphQL endpoint returned an error.

S3_ERROR

AWS S3 returned an error.

FATAL_ERROR

Unexpected error encountered. This could be a result of client or backend bug and unlikely to be user recoverable.

POLICY_ERROR

The API call failed due to the user having insufficient entitlements.
